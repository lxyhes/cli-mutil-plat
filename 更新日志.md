## v0.7.1 (2026-04-08)

### 新功能
- **社区生态**：一键发布 Skill/MCP/工作流/团队到 SpectrAI 社区，支持 deep link 安装和自动登录关联
- **项目管理**：新增需求管理、迭代管理（Sprint）、团队看板模块，支持指派、优先级、评论附图
- **Mission v2 并行引擎**：任务级并行执行、独立规划模型和执行 Provider 选择、全生命周期进度透出
- **Provider 管理升级**：3列分类布局、收藏置顶、新建会话分类筛选、模型切换刷新按钮
- **AI 管家增强**：记忆系统与主动互动、L3 自动处理事件、支持独立悬浮窗口
- **新手引导**：任务驱动模式引导 Tour 引擎，6 个 UI 组件
- **MCP OAuth**：自包含 OAuth 授权流程（浏览器 + 本地回调 + 持久化）
- **工作流增强**：mission 步骤类型、Skill 智能推荐、默认工作目录继承、Sprint 开发冲刺模板
- **AI 模型配置**：分组显示（内置 CLI / API 中转 / 自定义）
- **全局 Toast 通知系统**
- **跨平台增量升级支持**（Win + Mac）

### 优化
- 升级核心依赖 claude-agent-sdk / gemini-cli / codex
- SessionStore 切换为 vanilla store + 自定义 hook
- Dashboard Live Monitor 分批加载 + 固定面板高度
- 团队 Reconciler 简化为会话状态驱动模型
- 渲染进程内存泄漏修复（7 项）
- Worktree 合并后自动清理目录和分支

### 修复
- 修复 Codex 会话超时和消息丢失问题
- 修复团队模式 Leader 死锁、成员卡死等多个严重 Bug
- 修复第三方模型上下文溢出死循环
- 修复 MCP 工具名前缀丢失兼容问题
- 修复会话恢复时 spectrai-agent MCP 不注入
- 修复工作流编辑器多个前端 Bug
- 修复消息队列中断关键词匹配过于宽泛
- 修复忘记密码接口未提示邮箱未注册
